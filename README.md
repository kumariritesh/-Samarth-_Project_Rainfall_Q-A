# Project Samarth – Rainfall Q&A Prototype

This project compares rainfall data between Madhya Pradesh and Thane, Maharashtra using official datasets from data.gov.in.  
Built using Python and Google Colab, it answers rainfall-related questions interactively.

## Datasets
- Madhya Pradesh Rainfall Data (2011–2014)
- Thane District Rainfall Data (Daily)

## Tools Used
- Python (Pandas)
- Google Colab
- data.gov.in datasets
